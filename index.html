github.com/ece-berrak/aeo-content-pipeline 04 / 05
AUTOMATION · AI-NATIVE · OPEN SOURCE

Brand-aware content draft generator, Notion + n8n + Claude

A Notion-driven, n8n-orchestrated pipeline that turns a single keyword ticket into a fully drafted, voice-aligned long-form blog (or LinkedIn post, X thread, glossary entry) inside a Google Doc — ready for human review and manual publishing. Reads a 16-section Brand Context page, calls Claude Sonnet 4.6 with prompt caching enabled, drops the draft into Drive and writes the link back to the Notion ticket.

  • ~$0.02 per draft with prompt caching (90% input cost reduction)
  • 12-node n8n flow, 3 APIs (Notion, Anthropic, Google Docs)
  • 10 content formats supported via Brand Context format specs
  • Open-sourced under MIT
Open source github.com/ece-berrak/aeo-content-pipeline →

Research agent on n8n, $0/month

  • $0/month — runs entirely on free tiers (n8n, Gemini, Tavily, Notion)
